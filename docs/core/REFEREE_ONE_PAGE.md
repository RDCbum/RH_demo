# Referee one-page (index)

This is a one-page index for the ERURH demo, intended to minimize jumping
between files. It points to the current capsule and the paper/preprint.

## Core claim (conditional, hybrid)
The Lean core formalizes the logical chain to RH. The analytic lemmas (A1/A2,
A/B/C) are proved in the paper + preprint, and numeric certificates are
validated by the gate. Under these analytic inputs and the verified numeric
artifacts, the Lean chain yields RH for `xiAlpha`.
